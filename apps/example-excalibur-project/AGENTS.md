This is an Excalibur.js game project.

Visit the [excalibur documentation](https://excaliburjs.com/docs/) for information about engine setup, actors, scenes, resources & input handling.

The entry point is `src/main.ts` which creates the Excalibur engine, starts the loader, and adds the Tiled map to the scene

## Misc

- `res/tileset.tsx` and `res/tileset.tssx` are Tileset files (and not typescript files)
